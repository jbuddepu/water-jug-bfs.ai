# Water Jug Problem using BFS

Solves the classic water jug problem using Breadth-First Search (BFS) algorithm.

## Problem
Given 4 jugs with capacities [12, 8, 3, 5] liters and an unlimited water supply, find a sequence of moves to reach the goal state of (6, 6, 0, 0).

## Files
- `water_jug_bfs.py` - Python implementation using BFS
- `water_jug_bfs.pl` - Prolog implementation using BFS

## Usage
```bash
python water_jug_bfs.py
```

## Operations
- Fill a jug to capacity
- Empty a jug
- Pour from one jug to another

## Output
Prints the solution path showing all state transitions to reach the goal.
